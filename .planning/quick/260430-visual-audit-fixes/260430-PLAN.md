## Task 5 (M-01): Sidebar colapsável — slide-out drawer overlay

### 5a: useUIStore.ts — adicionar estado do sidebar

**File:** `torre-de-controle/src/stores/useUIStore.ts`

Remover o comentário que impede isSidebarOpen e adicionar ao UIState:

```typescript
isSidebarOpen: boolean
setSidebarOpen: (open: boolean) => void

```

Valores iniciais:
```typescript
isSidebarOpen: false,
setSidebarOpen: (open) => set({ isSidebarOpen: open }),
toggleSidebar: () => set((s) => ({ isSidebarOpen: !s.isSidebarOpen })),
```

### 5b: AppSidebar.tsx — transição CSS + overlay

**File:** `torre-de-controle/src/app/layout/AppSidebar.tsx`

Substituir o `<nav>` atual por um fragmento com overlay + sidebar com transição:

```tsx
import { useUIStore } from '@/stores/useUIStore'
import { useEffect } from 'react'

export function AppSidebar() {
  const { isSidebarOpen, setSidebarOpen } = useUIStore()
  
  // Fechar com Escape
  useEffect(() => {
    const handler = (e: KeyboardEvent) => { if (e.key === 'Escape') setSidebarOpen(false) }
    document.addEventListener('keydown', handler)
    return () => document.removeEventListener('keydown', handler)
  }, [setSidebarOpen])

  return (
    <>
      {/* Overlay — só visível abaixo de lg quando sidebar aberto */}
      <div
        className="lg:hidden fixed inset-0 bg-black/40 z-40 transition-opacity duration-200"
        style={{
          opacity: isSidebarOpen ? 1 : 0,
          pointerEvents: isSidebarOpen ? 'auto' : 'none',
        }}
        onClick={() => setSidebarOpen(false)}
      />

      {/* Sidebar */}
      <nav
        style={{
          position: 'fixed',
          top: '12px',
          left: '12px',
          width: '250px',
          height: 'calc(100vh - 24px)',
          background: 'var(--sidebar)',
          borderRadius: '1rem',
          boxShadow: '0 0 2rem 0 rgba(136, 152, 170, 0.15)',
          display: 'flex',
          flexDirection: 'column',
          overflowY: 'auto',
          zIndex: 50,
          transition: 'transform 0.25s ease',
        }}
        className="translate-x-[-280px] lg:translate-x-0"
        style-override-for-open={isSidebarOpen ? { transform: 'translateX(0)' } : undefined}
      >
        {/* ... conteúdo idêntico ao atual ... */}
      </nav>
    </>
  )
}
```

**IMPORTANTE:** A classe Tailwind `translate-x-[-280px]` não funciona com CSS variables/style inline ao mesmo tempo. Usar um atributo `data-open` + CSS via style inline é mais confiável:

```tsx
<nav
  style={{
    position: 'fixed',
    top: '12px',
    left: '12px',
    width: '250px',
    height: 'calc(100vh - 24px)',
    background: 'var(--sidebar)',
    borderRadius: '1rem',
    boxShadow: '0 0 2rem 0 rgba(136, 152, 170, 0.15)',
    display: 'flex',
    flexDirection: 'column',
    overflowY: 'auto',
    zIndex: 50,
    // Abaixo de lg: off-screen por default, desliza quando aberto
    transform: isSidebarOpen ? 'translateX(0)' : undefined,
    transition: 'transform 0.25s ease',
  }}
  className="max-lg:translate-x-[-280px]"
>
```

Usar a classe Tailwind v4 `max-lg:` (equivalente a `@media (max-width: 1023px)`) para o estado fechado padrão. Quando `isSidebarOpen=true`, o style inline `transform: translateX(0)` sobrescreve.
